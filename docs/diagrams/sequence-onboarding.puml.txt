@startuml sequence-onboarding
  title Fluxo de Registro + Onboarding - FitPlanner
  
  actor Cliente
  participant "API Gateway" as GW
  participant "user-service" as US
  
  Cliente -> GW: POST /api/users/register\n{name, email, cpf, password}
  GW -> US: forward request
  US -> US: Cria usuário + hash senha\nGera JWT
  US --> GW: 201 Created {token}
  GW --> Cliente: 201 Created {token}
  
  == Onboarding (primeiro acesso) ==
  
  Cliente -> GW: POST /api/users/me/onboarding\n{age, gender, height_cm,\nweight_kg, activity_level}
  GW -> US: forward (JWT validated)
  US -> US: Salva UserProfile\nonboarding_completed = true
  US --> GW: 200 OK {profile}
  GW --> Cliente: 200 OK
  
  note right of Cliente
    Após o onboarding o cliente
    é redirecionado para o dashboard
    onde ele mesmo cria seus treinos,
    registra nutrição e define metas.
  end note
  
  @enduml
